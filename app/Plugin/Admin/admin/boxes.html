With Progressbar

Progressbar's width is 150px

By default, a progress bar in a widget box's header will be 150px wide, but you can always change this by putting an inline style on the progress bar.

<div class="progress" style="width: 200px; "></div>

Sliders

Creating a slider requires jQuery-UI

Just create an element on a widget box's header toolbar, and initialize jquery-ui slider on that element. Easy!

Even Switches!

iOS Style switches

Creating switches is very easy!
Just put a checkbox or radio button inside a widget box's header, and give it the attribute

data-provide="ibutton"

Note: This requires the iButton plugin, do not forget to include it on your page

With Collapsible Form Toolbar

How to do this?

With the help of Bootstrap's Collapse plugin, you do not need to write any javascript code!

  1. Create a toolbar with an id attribute your-toolbar-id
  2. Create a button with data-toggle="collapse" data-target="#your-toolbar-id"
  3. Give the toolbar the class attribute collapse in
  4. Done!

Note: If you want to have the toolbar collapsed by default, remove the class in from the toolbar
